# **App Name**: Startify

## Core Features:

- Basic Layout: Implement a simple layout with a header, main content area, and footer.
- Navigation Bar: Create a basic navigation bar with links to different sections of the page.
- Content Section: Add a content section where you can add text and images.
- Theming support: Toggle between light and dark mode using local storage.

## Style Guidelines:

- Background color: Light grey (#F0F0F0) for a clean, minimal look.
- Primary color: Blue (#3498DB) for main elements and links.
- Accent color: Orange (#E67E22) for call-to-action buttons.
- Font: 'Inter', a sans-serif font for clean and modern readability. Use for both headings and body text.
- Use a simple, grid-based layout for easy content arrangement.
- Use simple, outline-style icons for a minimalist design.
